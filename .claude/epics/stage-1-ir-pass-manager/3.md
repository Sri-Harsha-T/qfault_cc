# Task: 1-D-2 — GitHub Actions CI matrix

## Description

Set up a GitHub Actions workflow that builds and tests QFault on both supported compilers (clang-17 and gcc-13) in debug and release configurations, triggered on every push and pull request to `main`.

## Acceptance Criteria

- [ ] `.github/workflows/ci.yml` created
- [ ] Matrix: `{compiler: [clang-17, gcc-,14]} × {build_type: [debug, release]}`
- [ ] Each matrix job runs: `cmake --preset $build_type && cmake --build ... && ctest ...`
- [ ] CI passes on the current (empty sources) state — green from the start
- [ ] ASAN build runs as a separate job (not matrix — slow, single run on push to main only)
- [ ] Workflow uses `ubuntu-22.04` runner
- [ ] Build artefacts cached via `actions/cache` on the CMake FetchContent directory

## Technical Details

- Install clang-17 via `apt-get install clang-17 lld-17`
- Install gcc-13 via `apt-get install gcc-13 g++-13`
- Use `CC=clang-17 CXX=clang++-17` / `CC=gcc-13 CXX=g++-13` environment variables
- Cache key: hash of `CMakeLists.txt` + `CMakePresets.json`

## Dependencies

- Task 001 (CMakeLists.txt) must exist before CI can reference presets
