# Task: 1-D-4 — scripts/quick-test.sh

## Description

Fill in `scripts/quick-test.sh` so it performs a fast pre-commit validation: configure + build + test in the debug preset, total time under 60 seconds. Also fill in the remaining stub scripts (`full-build.sh`, `asan-ubsan.sh`, `compare-stim.sh`).

## Acceptance Criteria

- [ ] `scripts/quick-test.sh` builds the debug preset and runs `ctest -j` — completes in <60s
- [ ] `scripts/quick-test.sh` exits non-zero if any test fails
- [ ] `scripts/full-build.sh` builds both clang and gcc release presets
- [ ] `scripts/asan-ubsan.sh` builds the asan preset and runs all tests under sanitizers
- [ ] `scripts/compare-stim.sh` is a stub that prints "Stim oracle: not yet implemented" and exits 0 (will be filled in Stage 3)
- [ ] All scripts are executable (`chmod +x`)
- [ ] `./scripts/quick-test.sh` runs successfully in CI (no extra dependencies)

## Technical Details

- `quick-test.sh` should skip reconfiguration if `build/debug` already exists (use `-N` or check)
- Print elapsed time at the end: `echo "Done in ${SECONDS}s"`
- Use `set -euo pipefail` at the top of every script

- Task 001 (CMakeLists.txt) must exist for scripts to invoke cmake
